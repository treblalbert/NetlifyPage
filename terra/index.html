About Terra

Terra is a voxel-based sandbox game created by Albert Freeman with no limits - literally, there is no height limit. Currently, the game is in the early prototype stage.

Current Features

  • Procedural generation (seed-based, with super trees and caves)
  • Volumetric clouds
  • Day/night cycle
  • Save/load system (partially implemented, currently being improved)
  • First and third person perspectives
  • Basic creative building and inventory with 10+ block types

Planned Features

  • Multiplayer (currently in development)
  • Unique survival experience
  • More world customization options (island size, generation parameters, day/night duration, etc.)
  • Additional block types
  • In-game coding and modding support
  • Steam page
  • Performance optimization

About Albert Freeman

Hi! I'm Albert Freeman, a 27-year-old from Barcelona. I've had many attempts at creating video games, with several in the realm of voxel worlds. Terra is meant to capture what OG Minecraft was all about - FREEDOM AND NO CORPORATE GREED.

I'm not a professional game developer and have never worked in the industry, but I believe I'm proficient enough in all the required areas to create a game. So here's another attempt! I hope you love it as much as I enjoy making it. Please enjoy! <3
