About Terra

Terra is a voxel-based sandbox game created by Albert Freeman with no limits - literally, there is no height limit. Currently, the game is in the early prototype stage.

In its current form, Terra is an island-based survival experience where players find themselves stranded on a procedurally generated island. The goal is to create a unique survival-packed experience that captures the feeling of being stranded on an island, with future plans to expand into multiplayer gameplay.

As development progresses, Terra will evolve into a floating islands-based survival experience, where the world will be made up of multiple sky islands, focusing on exploration, survival, and building across floating lands. This update will mark a major shift in world design and gameplay style.

Development Roadmap

This roadmap outlines the current goals for Terra's early prototype development. These plans may evolve as development progresses.

Prototype v1.3.0

  • Gameplay & Camera: Improve the third-person camera for smoother control and better angles
  • Building System: Allow block rotation with an R key rotation animation
  • Player Customization: Start with a random player skin, then implement a Skin Selector menu
  • Animal System v1: Add Animal Behavior (basic AI), implement Animal Spawning logic, introduce the first animal: Pig
  • New Blocks: Stairs, Door (with interaction system)
  • Environment: Add foliage for visual variety
  • World Improvements / Fixes: Fix blocks that can't be broken, fix terrain saving issues

Prototype v1.4.0

  • Core Gameplay: Convert the game into Survival Mode
  • Inventory System: Modify the block system so ownership is quantity-based
  • New Features: Add "The Crafter" for crafting items, add "Press Q to throw cube" mechanic (usable with The Crafter)
  • Combat & NPCs: Add health systems for both player and NPCs, add a weapon system for combat
  • Events: Add an invasion system — plane-based, with random timing spawns
  • Rest & Time: Add bed and sleeping mechanics, implement a clock/time system

Prototype v1.5.0

  • Survival Enhancements: Add a hunger system, add hunting and cooking features, introduce "The Cooker" for preparing food

Tree Update (Future Update)

  • Nature Systems: Implement tree falling physics, add automatic tree spawning, add manual tree planting and regrowth

Future Update – Sky Islands

  • The game will evolve into a floating islands-based survival experience
  • The world will be made up of multiple sky islands
  • Focus on exploration, survival, and building across floating lands
  • This update will mark a major shift in world design and gameplay style

Prototype v1.6.0

  • Multiplayer: Implement multiplayer functionality for co-op gameplay

About Albert Freeman

Hi! I'm Albert Freeman, a 27-year-old from Barcelona. I've had many attempts at creating video games, with several in the realm of voxel worlds. Terra is meant to capture what OG Minecraft was all about - FREEDOM AND NO CORPORATE GREED.

I'm not a professional game developer and have never worked in the industry, but I believe I'm proficient enough in all the required areas to create a game. So here's another attempt! I hope you love it as much as I enjoy making it. Please enjoy! <3
