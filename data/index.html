Python Libraries

Pandas

Library

A powerful data manipulation and analysis library providing data structures like DataFrames for working with structured data.

import pandas as pd
df = pd.read_csv('data.csv')
df.head()

NumPy

Library

Fundamental package for scientific computing with support for arrays, matrices, and mathematical functions.

import numpy as np
arr = np.array([1, 2, 3, 4, 5])
mean = np.mean(arr)

Matplotlib

Library

A comprehensive library for creating static, animated, and interactive visualizations in Python.

import matplotlib.pyplot as plt
plt.plot([1, 2, 3, 4])
plt.ylabel('Some Numbers')
plt.show()

Seaborn

Library

A statistical data visualization library based on matplotlib that provides a high-level interface for attractive graphics.

import seaborn as sns
sns.histplot(data=df, x='column_name')

Scikit-learn

Library

A machine learning library featuring various classification, regression and clustering algorithms.

from sklearn.linear_model import LinearRegression
model = LinearRegression()
model.fit(X_train, y_train)

Data Manipulation

DataFrame

Data Structure

A two-dimensional labeled data structure with columns of potentially different types, similar to a spreadsheet or SQL table.

# Creating a DataFrame
data = {'Name': ['John', 'Anna', 'Peter'], 'Age': [28, 22, 35]}
df = pd.DataFrame(data)

GroupBy

Operation

A process that involves splitting data into groups, applying a function to each group, and combining the results.

# Group by category and calculate mean
df.groupby('category')['value'].mean()

Merge/Join

Operation

Combining DataFrames based on common columns or indices, similar to SQL JOIN operations.

# Merge two DataFrames on a common column
merged_df = pd.merge(df1, df2, on='key_column')

SQL Fundamentals

SELECT Statement

Query

Used to select data from a database. The result is stored in a result table, called the result-set.

SELECT column1, column2
FROM table_name;

WHERE Clause

Clause

Used to filter records and extract only those that fulfill a specified condition.

SELECT * FROM Customers
WHERE Country='Mexico';

JOIN

Operation

Used to combine rows from two or more tables, based on a related column between them.

SELECT Orders.OrderID, Customers.CustomerName
FROM Orders
INNER JOIN Customers ON Orders.CustomerID = Customers.CustomerID;

GROUP BY

Clause

Groups rows that have the same values into summary rows, often used with aggregate functions.

SELECT COUNT(CustomerID), Country
FROM Customers
GROUP BY Country;

Window Functions

Function

Perform calculations across a set of table rows that are somehow related to the current row.

SELECT
name,
salary,
RANK() OVER (ORDER BY salary DESC) as salary_rank
FROM employees;

Advanced SQL

CTE (Common Table Expression)

Concept

A temporary named result set that you can reference within a SELECT, INSERT, UPDATE, or DELETE statement.

WITH RegionalSales AS (
SELECT region, SUM(amount) AS total_sales
FROM orders
GROUP BY region
)
SELECT * FROM RegionalSales;

Subquery

Concept

A query nested inside another query, used to return data that will be used in the main query.

SELECT product_name
FROM products
WHERE product_id IN (
SELECT product_id
FROM order_details
WHERE quantity > 100
);

Power BI Fundamentals

Data Model

Concept

The foundation of a Power BI report, consisting of tables and relationships that define how data connects.

Creating relationships between fact tables (transactions) and dimension tables (products, customers, dates).

DAX (Data Analysis Expressions)

Language

A formula language used in Power BI, Power Pivot, and Analysis Services for creating custom calculations.

Total Sales = SUM(Sales[Amount])
YoY Growth = [Total Sales] - CALCULATE([Total Sales], SAMEPERIODLASTYEAR('Date'[Date]))

Measures

Concept

Calculations created using DAX that are computed at query time based on user interaction with reports.

Total Revenue = SUM(Sales[Revenue])
Average Sale = AVERAGE(Sales[Amount])

Calculated Columns

Concept

Columns added to a table in the data model using DAX formulas, computed during data refresh.

Full Name = Customer[First Name] & " " & Customer[Last Name]
Profit Margin = (Sales[Revenue] - Sales[Cost]) / Sales[Revenue]

Power BI Visualization

Slicers

Visual

Interactive filters that allow users to segment and filter the data in a report.

Adding a slicer for "Year" or "Region" to allow users to filter the entire report.

Drillthrough

Feature

A feature that allows users to right-click a data point and navigate to a detailed page focused on that specific item.

Setting up a product details page that users can access by right-clicking a product in a visualization.

Bookmarks

Feature

A way to capture the current state of a report page, including filters and visual interactions.

Creating bookmarks for different views of the same data to create an interactive storytelling experience.
