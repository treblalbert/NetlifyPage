Python Libraries

Pandas

Library

A powerful data manipulation and analysis library providing data structures like DataFrames for working with structured data.

import pandas as pd
df = pd.read_csv('data.csv')
df.head()

Use Case: Data Cleaning

Pandas is essential for handling missing values, filtering data, and transforming datasets before analysis.

NumPy

Library

Fundamental package for scientific computing with support for arrays, matrices, and mathematical functions.

import numpy as np
arr = np.array([1, 2, 3, 4, 5])
mean = np.mean(arr)

Use Case: Numerical Operations

NumPy provides efficient array operations and mathematical functions that form the foundation for many data science libraries.

Matplotlib

Library

A comprehensive library for creating static, animated, and interactive visualizations in Python.

import matplotlib.pyplot as plt
plt.plot([1, 2, 3, 4])
plt.ylabel('Some Numbers')
plt.show()

Use Case: Custom Visualizations

Matplotlib allows for complete control over plot aesthetics and is ideal for creating publication-quality figures.

Seaborn

Library

A statistical data visualization library based on matplotlib that provides a high-level interface for attractive graphics.

import seaborn as sns
sns.histplot(data=df, x='column_name')

Use Case: Statistical Visualization

Seaborn simplifies creating complex visualizations like heatmaps, pair plots, and categorical plots with minimal code.

Scikit-learn

Library

A machine learning library featuring various classification, regression and clustering algorithms.

from sklearn.linear_model import LinearRegression
model = LinearRegression()
model.fit(X_train, y_train)

Use Case: Predictive Modeling

Scikit-learn provides tools for the entire machine learning workflow, from preprocessing to model evaluation.

Data Manipulation

DataFrame

Data Structure

A two-dimensional labeled data structure with columns of potentially different types, similar to a spreadsheet or SQL table.

# Creating a DataFrame
data = {'Name': ['John', 'Anna', 'Peter'], 'Age': [28, 22, 35]}
df = pd.DataFrame(data)

Use Case: Tabular Data Representation

DataFrames are the primary structure for data manipulation in pandas, allowing for intuitive data operations.

GroupBy

Operation

A process that involves splitting data into groups, applying a function to each group, and combining the results.

# Group by category and calculate mean
df.groupby('category')['value'].mean()

Use Case: Aggregated Analysis

GroupBy operations are essential for calculating summary statistics by categories in your data.

Merge/Join

Operation

Combining DataFrames based on common columns or indices, similar to SQL JOIN operations.

# Merge two DataFrames on a common column
merged_df = pd.merge(df1, df2, on='key_column')

Use Case: Data Integration

Merging allows you to combine data from different sources to create a comprehensive dataset for analysis.

SQL Fundamentals

SELECT Statement

Query

Used to select data from a database. The result is stored in a result table, called the result-set.

SELECT column1, column2
FROM table_name;

WHERE Clause

Clause

Used to filter records and extract only those that fulfill a specified condition.

SELECT * FROM Customers
WHERE Country='Mexico';

JOIN

Operation

Used to combine rows from two or more tables, based on a related column between them.

SELECT Orders.OrderID, Customers.CustomerName
FROM Orders
INNER JOIN Customers ON Orders.CustomerID = Customers.CustomerID;

Power BI Fundamentals

Data Model

Concept

The foundation of a Power BI report, consisting of tables and relationships that define how data connects.

Creating relationships between fact tables (transactions) and dimension tables (products, customers, dates).

Use Case: Data Relationship Management

A well-designed data model ensures accurate calculations and enables efficient filtering across related tables.

DAX (Data Analysis Expressions)

Language

A formula language used in Power BI, Power Pivot, and Analysis Services for creating custom calculations.

Total Sales = SUM(Sales[Amount])
YoY Growth = [Total Sales] - CALCULATE([Total Sales], SAMEPERIODLASTYEAR('Date'[Date]))

Use Case: Advanced Calculations

DAX enables complex time intelligence calculations, filtering contexts, and business metrics that go beyond standard aggregations.

Measures

Concept

Calculations created using DAX that are computed at query time based on user interaction with reports.

Total Revenue = SUM(Sales[Revenue])
Average Sale = AVERAGE(Sales[Amount])

Use Case: Dynamic Calculations

Measures respond to report filters and slicers, providing context-aware calculations that update with user interactions.
