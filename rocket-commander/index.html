About Rocket Commander

I created this game with the goal of teaching the basics of coding.

In this game, you'll go through 3 levels where you need to actually code each movement the rocket makes.

To make this possible, I built a super basic mini coding language inside Unity. It's designed to help beginners start understanding the logic behind coding.
